# 🎮 Game Hub by Shaifx

一个精美的 HTML5 游戏集合，采用现代化视觉设计，支持桌面和移动设备。

🔗 **在线试玩**: [https://0xshai.github.io/Game](https://0xshai.github.io/Game)

## 🎯 包含游戏

### 1. 🐝 蜜蜂射击 (Bee Shooter)
经典太空射击游戏的创意变体，玩家驾驶战机对抗入侵的蜜蜂军团。

**核心玩法**:
- 4 行蜜蜂阵型，金色→红色的渐进难度
- 蜜蜂会俯冲攻击，增加紧张感
- 60 秒限时挑战

**特色系统**:
- 🔫 **三连发子弹** - 扇形散射，覆盖更广
- 💀 **一击必杀** - 红色高能子弹，瞬间清场
- ❤️ **生命球** - 恢复生命值，续航保障
- 💣 **炸弹** - 小心躲避，触碰扣血

---

### 2. 🔨 打地鼠 (Whack-a-Mole)
奇幻森林主题的打地鼠游戏，精美画面与流畅手感完美结合。

**视觉亮点**:
- 🌲 沉浸式森林背景，昼夜渐变效果
- 🐹 4 种独特地鼠角色（普通/金/红/冰）
- ✨ 粒子特效与击中反馈

**游戏机制**:
- ⚡ 连击系统 - 连续命中获得分数加成
- 🎯 4 种地鼠类型，不同分值和特性
- ⏱️ 60 秒限时挑战最高分

---

### 3. ⚫⚪ 围棋 (Go)
现代科技风格的围棋对战平台，支持人机对弈。

**AI 难度**:
- 🎚️ **1-10 级可调难度** - 从入门到高手
- 🧠 智能算法，模拟真实对弈体验

**功能特性**:
- 📊 **领地计数** - 实时显示双方领地
- ↩️ **悔棋功能** - 支持无限悔棋
- 🎨 科技感棋盘设计，深色护眼模式

---

### 4. 🔵🔴 五子棋 (Gomoku)
赛博朋克风格的五子棋，经典玩法搭配炫酷视觉效果。

**视觉设计**:
- 💫 霓虹发光棋子，悬停预览
- 🎆 胜利连线动画效果
- 🌈 渐变色彩与粒子特效

**游戏模式**:
- 👥 双人对战
- 🤖 人机对战（AI 对手）
- 🎯 智能胜负判定
